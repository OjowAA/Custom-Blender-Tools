# Custom-Blender-Tools
This is a collection of geometry nodes and addons I have made for blender.
Feel free to try them out. Please note I am not activly developing any of these.
Bugs and errors are to be expected.

# Installing
### Adding custom Geometry nodes to blender
The blender files contain library assets. Simply add them to you assests folder (most likley "C:\Users\Name\Documents\Blender\Assets") and they will be availible in blender.

### Adding custom add-ons to blender
The standalone python files or zip files are addons. In the addon menu, you can select install from disk.

#### If further assistance is needed in downloading these tools please consult your nearest search engine or stackexchange.
